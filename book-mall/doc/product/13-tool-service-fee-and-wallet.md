## 1. 用户看到什么（当前口径）

### 课程 vs 工具

- **订阅中心 · 会员计划**：课程会员 **或** 工具会员订阅（发积分 / 工具准入）。
- **轻量包充值**：积分不足时购买加量包。

### 钱包 / 充值

- 充值进入 **积分池**，用于 **会员订阅扣费** 与 **按次生成扣积分**（平台代付用户）。
- BYOK 用户：厂商 API 自理；超额编排从轻量包扣积分。

### Gateway

- 使用 AI 工具前：Book 个人中心关联 Gateway Key；在 Gateway 绑定厂商凭证。
- 用量与 **预估厂商成本** 在 Gateway 控制台查看。

## 2. 典型流程

1. 注册 / 登录 Book，选择计费身份（平台代付 / 自带 Key）  
2. 开通 **会员订阅**（报价页）  
3. （BYOK）关联 Gateway → 绑定厂商 Key  
4. 打开 canvas / story / 工具站使用 AI  
5. 积分不足 → 购买 **轻量包**

## 4. 与旧规则的区别（对用户）

| 旧 | 新（积分换算 1.0） |
|----|-------------------|
| 须开通工具技术服务费月费 | 须 **会员订阅** |
| BYOK 单独收技术服务费 | **无**；会员准入 + 轻量包超额 |
| 每次生成扣点（Scheme A） | 平台代付：**按模型扣积分**；BYOK：厂商自理 |
