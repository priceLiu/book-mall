# 工具技术服务费与钱包（〇十三）

> **Gateway 流程**：[gateway-user-guide.md](./gateway-user-guide.md)

## 1. 用户看到什么

### 课程 vs 工具

- **订阅中心 · 会员计划**：只为 **课程/知识内容** 付费。
- **订阅中心 · 工具技术服务费**：为 **AI 工具** 按月开通；价格按工具不同（如试衣间约 3000 点/月，**以页面标价为准**）。

### 钱包 / 充值

- 充值进入 **钱包余额**，主要用于 **支付工具技术服务费**（开通或续订时扣一次）。
- **不是**「每生成一次扣一次」；生成时的云厂商费用由你在 **Gateway** 绑定的厂商 Key 自行承担。

### Gateway

- 使用 AI 工具前：Book 个人中心关联 Gateway Key；在 Gateway 绑定 KIE / 百炼 / DashScope 等厂商凭证。
- 用量与 **预估厂商成本** 在 Gateway 控制台查看。

## 2. 典型流程

1. 注册 / 登录 Book  
2. （可选）订阅 **课程** 会员计划 → 学习课程  
3. 充值钱包  
4. 在 **工具技术服务费** 页开通所需工具（按月）  
5. 关联 Gateway → 打开 tool-web / canvas / story 使用 AI  

## 3. 运营配置

- 各工具月费：`ToolServiceFeePlan`（后台 / seed；见 logic 文档定价表）。  
- 0 点月费表示免费套件（如费用明细页）。

## 4. 与旧规则的区别（对用户）

| 旧 | 新 |
|----|-----|
| 须充值过 + 余额线才进工具站 | 须 **开通对应工具月费** |
| 每次生成扣点 | **不**按次扣点；仅月费 + 自备厂商 Key |
| 课程订阅含工具 | **不含**；工具单独月费 |
