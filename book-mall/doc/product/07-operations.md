# 运营与公示（七）

1. **可配置项**：订阅价、最低余额线、前台预警线、大模型/工具单价、异常阈值等，宜在后台配置，避免为调价而改代码。  
2. **扩展**：会员活动、优惠券等可在用户中心增入口，需与 **计费/余额** 规则一致。  
3. **用户知情权**：在销售页、会员中心、产品详情中 **清晰公示**：  
   - 订阅与余额的 **边界**；  
   - **最低余额线** 及对使用的影响；  
   - **按量计费** 的大致方式；  
   - **退款政策**，含 **应扣未扣优先结算**；  
   - **订阅退款** 若与余额退款不同，须 **分别说明**。  
4. **法务合规**：实际条款以法务审定为准；产品文档与前台文案需 **同步更新**。
