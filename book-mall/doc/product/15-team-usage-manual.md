# 团队 / 公司版 使用手册

> 面向对象：团队主账号（OWNER）、管理员（ADMIN）、普通成员（MEMBER）。
> 配套设计：`doc/product/14-tenant-team-design.md`。
> 说明：本手册按目标态编写；带「（开发中）」标记的入口待对应轨道上线后可用。

---

## 0. 一分钟看懂

- **一份套餐，按席位买，全员共享积分池**：团队买多少席位决定价格；积分进「团队共享池」，谁生成谁扣，主账号可给每人设上限。
- **资源共享**：团队有「公共素材库」（大家可用）+「个人私有库」（只有自己看）。
- **账单透明**：团队总账看整体收支，成员明细看「谁、用了什么模型、花了多少积分」，失败/取消自动返还。
- **账号互通**：一次登录，Canvas / Story / 电商 / 工具站 / 提示词全站可用（单点登录）。

## 2. 主账号：开通团队

1. 打开**报价页** `/pricing` → 切到「**团队 / 公司**」→ 选「按月 / 按年」。
2. 选档位（标准 / 进阶 / 高级 / 豪华 / 至尊）。**黑色**是套餐自带席位与积分；**紫色**是增购席位的每席单价（人越多每席越便宜）。
3. 选择**席位数**（= 同时可生成的成员数），系统按「基础套餐 + 增购席位」算总价。
4. 下单付款 → 自动创建团队、生成**共享积分池**、你成为主账号并占 1 个席位。

> 价格示例（首版，可后台调整）：团队·标准 ¥199/月含 3 席、每席 1500 积分；增购席位 1–2 人 ¥59/席、3–5 人 ¥49/席、6+ 人 ¥39/席。

## 3. 主账号 / 管理员：成员与席位

入口：`个人中心 → 团队管理`（开发中）。

- **邀请成员**：
  1. 「邀请成员」→ 填写对方**手机号**，系统发送短信（验证码 + 邀请链接）。
  2. 对方点链接 → 用 Book 账号登录/注册 → 接受邀请 → 自动占用一个空席位。
- **分配 / 回收席位**：席位有限（= 已购席位数）。回收某成员席位后，其无法再用团队池生成（仍可看自己历史）。
- **设角色**：把信任成员设为「管理员」，可帮你管成员/Key（但不能续费充值）。
- **设每人上限**：给某成员设「本月最多用 N 积分」，防止单人用超。

> 想加人但席位满了？到报价页**增购席位**，价格按当前人数所在席位带计算。

---

## 4. 成员：加入与使用

1. 收到邀请链接/邮件 → 用 Book 账号登录 → 接受邀请。
2. 进入任意工具（Canvas / Story / 电商 / 工具站），顶部把「当前空间」切到**团队空间**。
3. 正常生成：消耗的是**团队共享池**积分（不是你个人的）。
4. 在 `个人中心 → 积分用量中心` 查看自己的每次生成与扣分。

> 个人空间和团队空间是分开的：个人空间用你自己的积分与私有资产；团队空间用团队池与团队资源。切换不影响对方。

## 5. 资源共享：公共库 vs 私有库

| 库 | 谁能看 | 谁能删/改 |
|---|---|---|
| **团队公共库** | 全体成员 | 仅主账号 / 管理员 |
| **个人私有库** | 仅本人 | 仅本人（成员离队时主账号可处置） |

- 把好作品「**设为公共**」即进公共库，全队可复用（如统一风格图、模板、参考视频）。
- 删除公共素材会**二次确认**，并提示「不可恢复，含云端存储（OSS）」。
- 成员离开团队时，主账号可选择把其私有资产「转公共 / 移交他人 / 删除」。

## 6. 账单与对账

### 6.1 主账号 / 管理员：团队总账（开发中入口 `团队管理 → 账单`）
- **积分账单（按月）**：本月发放、消耗、返还、充值、期末剩余，并**按模型**拆分。
- **按成员/席位下钻**：谁花得多一目了然，可导出 CSV。
- **BYOK 账单**（若团队自带厂商 Key）：技术服务费（按席位）+ 资源费（云存储 / 出网流量 / 任务数）。

### 6.2 成员：我的用量
- `个人中心 → 积分用量中心`：余额（团队空间下为团队池）、按模型聚合、最近 50 条生成明细（时间、模型、类型、来源、积分、状态）。
- 失败 / 取消的生成**不扣分**（自动返还），账单可见。

### 6.3 一次生成包含哪些信息
每次生成都会记录：时间、操作人、席位、应用来源、模型、秒数/张数、消耗积分、计费方式（积分/BYOK）、状态。便于团队核对「钱花在哪、谁花的」。

## 7. API Key（团队自带厂商 Key · BYOK）

> 适合已有阿里/火山/KIE 等厂商 Key 的团队：模型费用与厂商直接结算，平台只收技术服务费 + 资源费，不扣积分、不赚差价。

- 主账号 / 管理员在 Gateway 控制台为团队添加厂商 Key。
- **同一厂商可加多把 Key**（用别名/渠道区分，如「直采」「代理 A」）。
  - 现状：可添加、可存、可用别名区分；**「选默认 / 生成时指定用哪把 / 按渠道对账」正在开发（轨道 A）**。
- 生成时按团队默认凭证走；日志记录用了哪把 Key（渠道），供对账。

## 8. 登录与安全

- **单点登录**：一次登录，全站子应用互通。
- **多端在线 / 单会话**：当前同账号可多端同时在线；「一个席位同时仅一个活动会话（挤下线）」为可选安全项（开发中），开启后可防多人共用一个席位。

## 9. 常见问题 FAQ

**Q：团队积分用完了怎么办？**
A：主账号可续费/升级套餐或单独充值积分包；也可临时调低某些成员上限。

**Q：成员能看到别人的私有作品吗？**
A：不能。只有「设为公共」的素材全队可见。

**Q：成员能改套餐或充值吗？**
A：不能，仅主账号可操作资金类动作。

**Q：席位和并发是一回事吗？**
A：不是。席位是「计费 + 每人额度」口径；并发是团队同时可跑的任务数（按套餐档，标准 2 … 至尊 35）。

**Q：AI 课程也走积分吗？**
A：不走。AI 课程单独购买，不受积分限制。

**Q：怎么核对厂商账单？**
A：平台财务后台把厂商账单按标准模型名归口，与我们的逐次成本快照比对，输出差异表（多收/少收/缺失），团队侧只需看自己的积分与服务费账单。
