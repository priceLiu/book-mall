# 前台用户中心（四）

用户中心为 **登录后** 主入口；**游客无中心**。**免费注册用户**与游客在 **内容权限上一致**，但可使用本模块管理账号与空态业务数据。

## 4.1 个人信息概览（首页）

1. **基础信息**：头像、昵称、手机/邮箱、**当前身份**（游客链路不适用；免费未订阅 / 普通订阅 / 高级会员 **状态**）、订阅有效期。  
2. **核心数据**：**可用余额**、**最低余额线提示**（距最低线差额）、订阅剩余天数；**不再展示**「充值满额才能成为高级会员」类进度条，可改为「高阶/按量服务依赖余额与消费量」说明。  
3. **快捷入口**：续费订阅、充值、帮助与计费说明。  
4. **消息**：订阅到期、**余额预警**（多级）、产品更新、系统通知。

## 4.2 会员订阅管理

1. 当前套餐、生效/到期时间、自动续费开关。  
2. 续费、升年付、关自动续费、订阅订单列表。  
3. **权益说明**：普通订阅 vs **可使用高级型时的条件**（订阅 + 余额 ≥ 最低线 + 按量说明）。  
4. 历史订阅记录。

## 4.3 我的钱包

1. **余额**：可用、冻结（若有）、累计充值、累计消耗。  
2. **充值**：档位、支付方式（首期可 Mock）、到账状态。  
3. **明细**：充值、扣费、大模型/工具消耗、订单号、时间；**应扣未扣** 在结清前应有「待结算」展示（若无则须在帮助中说明结算延迟）。  
4. **提醒**：自定义 **余额过低** 提醒开关；与三册预警线联动。

## 4.4 我的知识产品

已学课程、进度、收藏、资料下载（能力范围仍受 **用户层级** 约束，见一册）。

## 4.5 我的工具产品

可用列表、使用记录、**大模型用量**（额度与消耗）、无权限/余额不足时的 **引导充值**。

## 4.6 订单管理

订阅订单、充值订单、消耗类订单；详情与凭证；异常反馈入口。

## 4.7 账户设置

资料、安全、消息开关、登录日志、**账号注销**（注销时 **余额须按二册提现规则结算**）。

## 4.8 帮助与反馈

FAQ（权限、计费、**提现与应扣未扣**）、意见反馈、客服入口。
