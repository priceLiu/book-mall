AI平台积分清零与退款规则公示方案（MD格式，可直接公示使用）
 
对标豆包官方规则制定，合规严谨，适配平台公告、用户协议、付费页面公示场景
 
一、总则
 
1. 积分定义
 
本平台积分是用于抵扣AI对话、文生图、视频生成、模型调用等功能消耗的虚拟服务权益，不具备法定现金价值，不可提现、转让、交易、折现，仅可在本平台内按规则使用。
 
2. 积分分类
 
平台积分分为三类，清零、退款规则相互独立：
 
1. 订阅套餐赠送积分：开通月度/季度/年度会员后，套餐内自动发放的周期型额度；
2. 单独充值积分：用户额外付费购买的算力积分包、加量包；
3. 活动免费积分：登录、签到、拉新、平台营销活动赠送的免费额度。
 
二、积分清零规则（核心公示条款）
 
（一）订阅套餐赠送积分（会员专属额度）
 
1. 清零周期：按会员计费周期清零（以用户开通会员当日为周期起始日，按月循环刷新），周期内未使用完毕的剩余积分，到期自动全额清零，不结转、不滚存至下一计费周期；
2. 退订规则：用户中途取消自动续费，当前计费周期剩余积分可正常使用至到期，到期后依旧清零，不折算为现金或余额；
3. 消耗优先级：系统默认优先消耗即将到期的订阅赠送积分，再消耗充值积分、免费积分。
 
（二）单独充值积分（付费预购算力包）
 
1. 有效期：自充值到账之日起，有效期12个月，有效期内积分可累积、叠加使用；
2. 清零规则：超出12个月有效期后，未使用的充值积分自动清零；会员到期不影响充值积分有效期，充值积分可独立使用至过期。
 
（三）活动免费积分
 
1. 每日登录免费积分：当日24:00自动清零，不可累积；
2. 拉新、节日等营销赠送积分：自到账之日起30天有效期，过期未使用自动清零；
3. 免费积分不参与任何退款、折现补偿。
 
三、退款规则（合规公示条款）
 
1. 订阅会员套餐退款规则
 
1. 7天冷静期全额退款：会员开通后7个自然日内，未消耗任何订阅赠送积分，用户可申请全额退还会员订阅费用；
2. 已消耗积分不予退款：冷静期内只要产生任意一次积分消耗（对话、生图、视频生成等），即视为服务已交付，剩余未使用积分不支持折算退款，订阅费用不予退还；
3. 周期内无理由不退：超出7天冷静期后，无论会员剩余积分多少，用户主动申请的无理由退款均不予受理；
4. 平台责任特殊退款：因平台服务器故障、服务中断、功能重大缺陷，导致用户无法正常使用服务的，可按未履约周期比例申请退费，或由平台统一补偿会员时长、赠送积分。
 
2. 单独充值积分退款规则
 
1. 常规无理由不退：充值积分包一经支付到账，无质量问题、非平台故障，不支持无理由退款；
2. 异常订单退款：出现重复扣款、积分未到账、充值金额错误等非用户原因的异常订单，用户可在72小时内提交订单凭证申请退款，核验属实后按原支付路径全额退还；
3. 已消耗部分不退：充值积分一旦产生消耗，剩余未使用额度不支持部分退款。
 
3. 通用退款排除情形
 
以下场景一律不支持退款：
 
1. 用户个人原因（用量预估错误、不想继续使用、体验预期不符等）申请退款；
2. 积分已过期清零、会员周期已结束；
3. 用户违规使用平台服务（刷量、盗刷、恶意调用模型）被封禁账号；
4. 活动赠送的免费积分、补偿积分，不具备退款资格。
 
四、补充说明（公示兜底条款）
 
1. 平台有权根据运营规则调整积分政策，规则变更将通过平台公告提前公示，变更前已获取的积分按原规则执行；
2. 积分清零、退款争议，用户可通过「设置-帮助与反馈-在线客服」提交凭证申诉，平台将在3个工作日内完成核实回复；
3. 本规则最终解释权归本平台所有，未尽事宜参照《用户服务协议》执行。
