文档说明
 
1. 全文无缺项、字段完整、格式固定，可直接用于AI工作流封装，交付编剧/分镜师/绘图/剪辑全岗位；
2. 分两大体系：【从零原创无原稿版】、【已有原稿翻新规整版】；
3. 生产单元固定10集一批，解决30/40/50/60集人设、剧情、视觉漂移问题；
4. 每一集强制输出10大固定模块，顺序不可调换、字段不可删减。
 
 
 
一、全局统一硬性强制规则（两套体系通用，AI全程遵守）
 
1. 三重永久锁死机制：文档首次生成后永久冻结，后续所有集数禁止修改
- 时代世界观锁：年代、地域、建筑、道具、色调统一，杜绝时代穿帮
- 人物视觉锁：五官、脸型、肤色、发型、穿搭、配饰、气质全程固定，禁止变脸、随意换装
- 剧情总纲锁：全集一句话总纲为唯一基准，剧情不得擅自增删、偏离
2. 批量生产规则：固定10集为1个生产单元，单次仅生成10集内容，不一次性批量产出更多集数
3. 单集叙事硬性标准：本集独立矛盾完整闭环，仅结尾设置1个跨集长线悬念；完整叙事结构：开场钩子→主线推进→支线互动→中段反转→本集矛盾收尾→预埋下集悬念
4. 输出强制要求：每一集严格输出10大固定模块，顺序固定、字段填满、禁止简写缺项；分镜时长精确到秒、分镜提示词使用固定拼接公式、视频渲染参数标准化
5. 批次校验规则：每完成10集必须输出篇章综合校验报告，筛查人设OOC、剧情偏离、视觉错乱、叙事残缺问题，并提供可落地修改方案

模块2：本集出场人物完整版视觉锁定复盘
 
每位出场人物统一输出12项内容，严格匹配全局人物档案，不可擅自改动：
 
1. 人物姓名、年龄、身高、体型体态
2. 固定脸型骨相：圆脸/方脸/鹅蛋脸/棱角轮廓/柔和轮廓
3. 永久五官细节：眼型、眼距、眉形、鼻梁高度、唇形、面部固定特征（痣/疤痕等永久不变元素）
4. 固定神态观感气质：清冷/温柔/凌厉/憨厚/忧郁/松弛/沉稳/叛逆
5. 固定皮肤质感：冷白皮/自然黄皮/小麦色/黝黑质朴肤色
6. 固定发型体系：日常发型、正式发型、发色、发丝软硬质感
7. 专属全套穿搭体系：人物专属主配色；居家/外出/职场/宴会固定穿搭款式
8. 永久固定配饰清单：项链、手链、手表、戒指、眼镜、帽子、背包，款式全程不更换
9. 本集临时穿搭微调说明（仅场景适配，不改动基础款式与配饰）
10. 本集人物情绪状态
11. 本集行为处事逻辑（贴合全局性格设定，禁止OOC）
12. 人物专属台词风格、口头禅
 
模块3：本集场景完整环境档案
 
单集所有出场场景逐条输出，每条场景包含：
 
1. 场景名称
2. 内景/外景
3. 时间区间：清晨/白日/黄昏/深夜
4. 年代装修布局、空间大小、陈设结构
5. 固定光影参数：主光色调、辅光明暗、整体亮度
6. 环境氛围：烟火气/冷清压抑/静谧松弛/紧张肃杀/暧昧柔和
7. 常驻固定摆放道具位置
8. 环境背景音效：细雨/城市车流/安静空寂/人群嘈杂/风声
 
模块4：本集道具精细化清单
 
1. 核心剧情道具：剧情作用、新旧质感、摆放/手持位置、年代合规性校验
2. 氛围辅助装饰道具：家具摆件、墙面装饰、桌面小件
3. 道具色彩、材质统一规范
4. 需要镜头重点特写的道具标注
 
模块5：本集结构化完整大纲（固定8要素逐条罗列）
 
1. 上一集承接剧情细节
2. 本集开场3秒抓人钩子画面与台词
3. 主线完整推进流程
4. 支线人物互动完整内容
5. 中段冲突、反转完整细节
6. 本集独立矛盾完整闭环解决过程
7. 本集新增长线伏笔埋设内容
8. 结尾指向次集的悬念完整描述
 
模块6：标准工业级影视剧本（固定排版格式）
 
统一单行排版结构：
 【场景序号】内外景+时间｜环境光影整体描述｜空间陈设与道具｜人物进场动作、肢体神态｜人物完整台词｜(括号标注人物当下情绪)｜本段剧情功能：铺垫/冲突/反转/收尾 
 
模块7：标准化分镜脚本表格（固定表头，无空白字段）
 
镜号 单镜头时长(秒) 景别(全景/中全景/中景/近景/特写/大特写) 镜头运动(固定/慢推/拉/左右横摇/跟随移动/环绕/慢移) 完整画面内容描述 人物动作、五官神态、穿搭配饰细节 画面同步台词/字幕 镜头整体情绪 适配BGM曲风 
 
模块8：分镜图AI生成提示词（强制固定拼接公式，顺序不可打乱）
 
固定拼接顺序：
 全剧统一固定画风词 + 全剧统一年代环境质感 + 人物完整五官脸型神态描述 + 人物本集固定穿搭+全套配饰 + 当前完整场景环境光影描述 + 本镜头景别+镜头运动参数 + 画面整体氛围关键词 + 材质细节质感描述 + 高清画质参数（8K、电影级、细腻光影） 
 
模块9：分镜视频成片统一渲染参数（剪辑直接落地）
 
1. 固定视频画幅：16:9 / 9:16（二选一全剧统一）
2. 全剧统一滤镜色调、饱和度、明暗对比度
3. 字幕规范：字体、字号、屏幕摆放位置、字幕底色
4. 镜头转场统一风格：淡入淡出/硬切/缓慢叠化
5. 全片节奏标准：文戏慢镜头时长、冲突戏紧凑镜头时长
6. BGM使用规则：文戏舒缓纯音乐、冲突戏紧张节奏配乐、结尾留白轻音乐
 
模块10：本集视觉&剧情综合校验报告
 
1. 人物校验：五官、发型、穿搭、配饰是否与全局档案完全统一，列出偏差项与修正方案
2. 场景校验：年代道具、光影色调是否统一，有无时代穿帮
3. 剧情校验：是否贴合全集总纲、有无人设OOC、单集矛盾是否完整闭环、伏笔是否前后呼应
4. 优化调整建议：薄弱戏份补充、节奏调整、镜头优化
