## 0. 已锁定的关键决策（务必遵守）

1. **不重做画布引擎与分格**：React Flow + Pro2 现有画布、节点排布、连线、分格一律复用，禁止重写。
2. **一切在 pro2 内**：不新建 `edition`；「剧本创作」入口做成 Pro2 模板/画布，不是新版本。
3. **节点 UI 一律 2.0 LibTV 样式**：人物设计 → `story-pro2-three-view`；风格 → 资产库 → 风格库 → `story-pro2-style-asset`；场景/道具/氛围/音效 → 资产库子菜单 + LibTV 媒体卡 + **Pro2InputDockShell 浮动 Dock**；分镜脚本 → **仅** `story-pro2-script-hub`（脚本生成器）；分镜视频 → `sbv1-video-engine`。
4. **全面切换到新工业化设计**：**废弃旧四表解析与数据结构**，解析与数据结构按 `docs/2.0 工业标准化脚本生产.md` 重做，不再适配旧版。
5. **长剧分批生成（方案 A）**：固定 10 集/批，首轮额外产 4 份永久冻结档案，次轮置顶冻结档案 + 已完成批次成品再续写。
6. **分镜图提示词中英双行**：模块 8 同时产「中文公式」+「`镜N(EN)：` 英文」；**生图用英文、展示用中文**。
7. **数据库**：画布 JSON 无需迁移；**仅**定稿导出 `SCRIPT_PACKAGE` 时需 **1 次 additive enum 迁移**（见 §5）。
8. **不影响分镜视频 1.0 (sbv1)**：版本识别 sbv1 在前、运行链未改；Pro2 改造与 sbv1 节点/资产 kind 无交叉。
