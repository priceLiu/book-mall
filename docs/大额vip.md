一、图片页面文字合规整改逐条梳理（结合预付费、虚拟积分、企业充值法规）
 
（一）高危违规表述（必须修改，存在监管风险）
 
1. 原文：积分永久有效、按月不清零 / 一次性预充，双池积分长期有效
风险点：
- 《单用途商业预付卡管理办法》《消费者权益保护法》明确：不得承诺虚拟储值类权益“永久有效”，永久二字属于绝对化极限词，市场监管可处罚；
- 企业大额预充值属于预付费服务，必须标注最长有效使用期限，不能写永久。
修改方案：
 
双池积分有效期为自充值到账起5年，有效期内无月度清零，到期未使用积分自动失效；平台可提前90天通知企业客户延期办理。
2. 原文：面值约¥227,272.72 / 面值约¥197,368.44（挂牌锚价估算）
风险点：
- 大额预充值支付金额是10万/20万/50万，却标注远高于实付的“面值”，极易被认定为变相赠送、返利、虚拟货币溢价，涉嫌违规预储值、变相融资；
- “面值”词汇容易被判定为代币、储值卡面额，国内禁止平台发行带面值的虚拟积分资产。
修改方案：
删掉全部「面值」字样，替换为：
 
积分算力等价服务参考价约XX元（仅算力市场价参考，不代表积分具备现金面值、不可兑现）
补充小字备注：积分仅为平台AI服务消耗额度，不具备现金价值，无法折现、赎回、提现。
3. 原文：企业大额预充、VIP大额套餐
补充约束文字，规避资金池风险：
增加备注：本预充值仅用于兑换平台AI模型调用服务，非理财、投资产品，无任何资金增值、返利收益。
 
（二）模糊、易产生纠纷的文字优化（公示必须写清楚）
 
1. 原文：席位与人均上限可自行调整；席位数量开通后可增购、自行分配

4. 原文：团队共享池合计XX通用+XX视频
