电商平台工具箱
要求:
1. 全屏显示
2. 设计相关的规范, 请根据 design 文件夹: COLORS.md（黑白蓝）、DESIGN.md、BUTTON.md、DIALOG.md（弹出层）、VIDEO.md（原生视频）、README.md
3. 功能描述:
电商主图制作
电商详情页制作
电商带货视频,视频动作, 穿搭视频,卡点跳舞换装, 视频运镜, 数字人, 户外对镜自拍, 爆款服装带货, 电商口播带货, 服装电商模特图.
IP 设计
海报制作
品牌 VI , 表情包, 
宣传片制作
广告短片

4. 项目架构, 部署, 与 canvas , tool, story 对齐, 需要在腾讯云平台自动构建与部署;
5. 该网站, 需要直接对外使用, 登录可以在 book-mall 登录. 个人中心, 有链接进入.
6. 该网站使用的模型计划方法, 有两种. 
a. 充值扣点, 按生成的张数,视频数, 秒, 来计算. 这个需要参考 云厂商挂牌价, 再 X 系数, 计算, 可以先定计费模板, 按每个生成来核算; 这个统一用我们gateway的管理员的 key , 但需要记录完整的用户使用情况, 或者是在 gateway ,申请一个可以区别用户的平台的 key , 主要是分开用量, 以便于核算. 
b. 充值扣点, 扣的是技术服务费, 用户自行去云厂商, 申请key , 在 gateway填写申请的key, 然后申请我们gateway的 key ,使用.
