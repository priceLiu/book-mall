# 漫剧 · 故事创作工作流（标准 · 唯一真源）

> **强制**：凡改动漫剧画布、大纲拆分、节点生成、连线、批量运行、多工作流隔离，**必须先读本文**。在新工作流规范发布之前，**不得**擅自改变阶段顺序、节点种类或「定稿生成工作流 = 仅拆分、不自动生成媒体」语义。

关联：[story-ops.md](./story-ops.md) · [design.md](./design.md) · [story-eng.md](./story-eng.md)

---

## 1. 一句话

**用户定稿故事大纲（大纲非空即可）→ 点「定稿生成工作流」拆分填入各节点 → 用户在各节点自行生成媒体。**

---

## 2. 产品共识（讨论结论 · 2025）

### 2.1 代码层 vs 操作层

| | 代码层 | 操作层 |
|---|--------|--------|
| 能力 | 主题 + 提示词可产出完整 Markdown（大纲/角色/分镜等） | 用户 **先确认大纲**，再 **定稿生成工作流** |
| 目的 | 模板与 LLM 上限 | **卡点在大纲**：未定稿不拆下游节点 |

### 2.2 定稿门槛（ intentionally 宽松）

- **硬门槛**：故事大纲 **非空** + 用户点击 **「定稿生成工作流」**（即确认并拆分）。
- **不强制**：角色、分镜、对白是否填满；无角色时角色列 **仍创建、可为空**。
- **不限制**「非正常用户」的用法：例如在分镜列手写 prompt 直接生成视频；工具面向懂行的快速用户，**不为空字段做过多拦截**。

### 2.3 定稿前 vs 定稿后

| 阶段 | 用户 | 系统 |
|------|------|------|
| **定稿前** | 可改大纲/角色/分镜/对白，保存，重跑 LLM | `scriptFinalized === false` |
| **定稿后** | 故事大纲 **只读**（审阅弹层不可保存/生成） | `scriptFinalized === true` |
| **删下游列** | 可回到定稿前：改大纲、再点「定稿生成工作流」 | 自动 `scriptFinalized = false`，starter 解除 `finalized` |

定稿后 **不可改 hub 文案**；若需大改，应 **删除本套媒体列**（或整套重来），回到阶段 A。

### 2.4 多工作流

同画布可存在 **多套** 或 **多条相同** 工作流，**互不影响**（`scriptHubId` / `hubNodeId` 隔离），除非用户 **手动改线**。

## 3. 阶段（不可打乱）

```text
阶段 A · 定稿前（故事主题 + 故事大纲）
  A1 输入主题
  A2 创作剧本 → 生成 / 编辑大纲（及可选角色、分镜）
  A3 审阅保存（定稿前可随时改）

阶段 B · 定稿生成工作流（故事大纲 ·「定稿生成工作流」）
  B1 用户确认 → scriptFinalized = true
  B2 创建节点链 + 拆分定稿文本到各列 rows
  B3 不 enqueue 三视图 / 分镜图 / 视频

阶段 C · 媒体（用户触发）
  各列单镜或底栏「全部生成」；剪映导出 ZIP
```

## 4. 固定节点链

| 顺序 | 类型 | 名称 | 职责 |
|------|------|------|------|
| 1 | `story-comic-starter` | 故事主题 | 主题、LLM、「创作剧本」 |
| 2 | `story-script-hub` | 故事大纲 | 定稿区 · **定稿生成工作流** |
| 3 | `story-character-column` | 三视图 | 拆分角色；**可无行** |
| 4 | `story-frame-column` | 分镜脚本 / 分镜图 | 拆分分镜 |
| 5 | `story-video-column` | 分镜视频 | 同步对白/脚本 |
| 6 | `jianying-export` | 剪映导出 | 顶对齐主题/大纲 |

连线：`主题 → 大纲 → 角色列 → 分镜列 → 视频列 → 剪映`。

---

## 5. 定稿 → 拆分映射

拆分前：`hubDataForColumnSync`。

| 定稿字段 | 目标 | 说明 |
|----------|------|------|
| `characterMd` / 大纲嵌入角色 | 角色列 rows | 无则空列 |
| `storyboardMd` / 大纲嵌入分镜 | 分镜列 rows | |
| 对白（分镜表 / 对白 Tab） | 分镜列 + 视频列 | 随分镜同步 |
| — | 剪映 | 导出时读视频列 runtime |

**禁止**：定稿后修改 hub；阶段 B 自动跑媒体；从分镜对白反推角色（除非定稿中已有角色段）。

---

## 6. 「定稿生成工作流」按钮

| 条件 | UI |
|------|-----|
| 大纲非空、未在跑、本套无三列 | **定稿生成工作流**（可点） |
| 已生成工作流 | **已定稿 · 工作流已生成**（禁用） |
| 用户删除本套媒体列 | 恢复可编辑 → 可再次定稿 |

数据字段：`StoryScriptHubNodeData.scriptFinalized`。
