# 漫剧画布 · 操作手册

## 1. 创作剧本

1. 拖入或载入 **漫剧启动** + **漫剧文案**（模板已预置连线）。
2. 填写主题，选择 LLM。
3. 点击 **创作剧本** → 自动连接「漫剧文案」并开始顺序生成大纲 → 角色 → 分镜。

## 2. 审阅与修改

1. 在 **漫剧文案** 节点点击 **审阅剧本**。
2. 弹出层切换：**故事大纲 / 角色设定 / 分镜脚本 / 对白**。
3. **编辑** Tab 修改 Markdown；**预览** Tab 查看渲染，可 **全屏 Word 式预览**。
4. 各段可单独 **生成 / 重新生成**；底栏可 **顺序生成全部文案**。
