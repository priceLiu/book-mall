# 漫剧画布 · 操作手册

在 **canvas-web** 画布内按步骤完成漫剧生产。本文面向日常使用；产品与开发细节见 [story-eng.md](./story-eng.md)。

---

## 1. 准备工作

### 1.1 新建画布

1. 打开 canvas-web → **新建项目**（或进入已有项目）。
2. 选择模板 **「漫剧全链路」**：初始只有 **漫剧启动** 与 **剪映导出** 两个节点。
3. 也可空白画布后，从顶部工具栏拖入 **漫剧启动**。

### 1.2 模型与 Provider

| 用途 | 配置方式 |
|------|----------|
| 故事大纲 / 角色 / 分镜文案（LLM） | **设置 → Provider**；或直接用 **系统 · DeepSeek**（服务端已配 `DEEPSEEK_API_KEY` 时出现），模型选 `deepseek-chat` |
| 三视图 / 分镜静帧（IMAGE） | 用户 Provider 或 **系统 · KIE** |
| 分镜视频（VIDEO） | **系统 · KIE** 等 |
| 对白配音（TTS） | OpenAI 兼容 TTS，如 `tts-1` |

**漫剧启动** 与各 Story 引擎 **文案 Tab** 内选 LLM；三视图、分镜图、视频、对白在各自弹窗 Tab 内 **独立选模型**。

### 1.3 顶部工具栏

- 顶部居中：节点类型图标（图片、文本、AI 引擎、漫剧启动、分镜图等）。
- 最右侧 **`>`**：**收到右侧**，变为竖条 icon 栏；竖条顶部 **`←`** 可展开回顶部。
- **`?`**：快捷键与操作说明。

## 2. 标准五步流程

```text
① 漫剧启动（文案链）
    ↓
② 角色设定 → 三视图
    ↓
③ 分镜脚本 → 分镜图（只创建节点，不生图）
    ↓
④ 各分镜图 · 镜N → 静帧 / 视频 / 对白
    ↓
⑤ 剪映导出
```

### 步骤 1 · 漫剧启动 · 生成三份文案

1. 点击 **漫剧启动** 节点。
2. 填写 **主题 / 概要**。
3. 选择 **LLM 模型**（三文案引擎共用）。
4. 点击 **开始生成（大纲+角色+分镜）**。
   - 自动在画布创建：**故事大纲 → 角色设定 → 分镜脚本**，并按顺序跑 LLM（避免并发限流）。
5. 等待三个节点状态均为 **已完成**（或分别打开各节点 **预览** Tab 查看 Markdown）。

**重跑文案**

- 已有三引擎时，按钮变为 **重新生成全部文案**。
- 需要忽略缓存时，点 **强制跳过缓存重跑**。
- 也可单独打开某引擎 → **文案** Tab → 编辑 Prompt → 生成 / 强制跳过缓存。

---

### 步骤 2 · 角色设定 · 三视图

1. 打开 **角色设定** 节点 → 底部 **三视图**（或弹窗 **三视图** Tab）。
2. 确认 **IMAGE 模型** 已选。
3. 勾选要生成的角色（可 **全选 / 全不选**）。
4. 点 **生成选中** 或 **生成全部**。
   - 为每个角色 **创建三视图节点**（同名已存在则跳过），并排入队列 **依次生图**。
5. **不必一次做完所有角色**；可按批次勾选。

**三视图「就绪」**（后续分镜图创建会检查）：

- 画布上有该角色的三视图节点；
- 状态 **已完成**，且已有预览图。

---

### 步骤 3 · 分镜脚本 · 批量创建分镜图

1. 打开 **分镜脚本** 节点 → 底部 **分镜图**。
2. 在 **分镜图** Tab 选择 **IMAGE 模型**。
3. 勾选镜号（悬停可查看该镜 **场景 / 画面 / 对白 / 视频提示** 完整内容）。
4. 点 **创建选中** 或 **创建全部**。

**规则说明**

| 情况 | 行为 |
|------|------|
| 该镜号 **尚无** 分镜图 | 创建新节点，写入 Prompt、对白、视频提示，连接三视图参考边 |
| 该镜号 **已有** 分镜图 | **不重复创建**；补写对白 / 视频提示，刷新 Prompt 与三视图连线 |
| 新建镜号 | 校验 **本镜出镜角色** 的三视图是否就绪；未就绪则整批提示，需先到步骤 2 补做 |
| 镜号都已存在、仅补对白 | **跳过三视图校验**，可直接 **创建选中** 补写 |

创建后 **不会自动生图**；需到各 **分镜图 · 镜N** 手动生成静帧。

---

### 步骤 4 · 分镜图 · 镜N · 静帧 / 视频 / 对白

点击分镜图节点底部四个按钮之一，打开操作弹窗：

| 按钮 / Tab | 作用 |
|------------|------|
| **分镜图生成** → **重新生成** | 编辑 Prompt、选 IMAGE 模型，生成静帧；点击生成后弹窗关闭 |
| **生成视频** | 查看视频提示，选 VIDEO 模型，生成或重跑本镜视频 |
| **生成对白** | 查看 **对白** 全文，选 TTS 模型，合成配音 |
| **视频+对白** | 同时配置 VIDEO + TTS，顺序触发 |

**对白 / 视频提示从哪来**

1. 优先读分镜图节点上已保存的字段；
2. 若为空，自动从 **分镜脚本** Markdown 表按镜号读取；
3. 打开 **生成对白** / **生成视频** 弹窗时，若节点缺字段会自动 **补写到节点** 并展示。

无对白文本（或仅为 `—`）时，**生成对白** 按钮不可用。

---

### 步骤 5 · 剪映导出

1. 打开 **剪映导出** 节点。
2. 选择 **分镜包 ZIP**（推荐）或 **剪映草稿 ZIP**。
3. 下载后含各镜视频 / 音频及 **全集.srt**（字幕来自分镜表对白列）。

---

## 3. 删除节点后如何再次生成

删除自动创建的节点后，可以再次生成；入口如下。

### 3.1 故事大纲 / 角色设定 / 分镜脚本（漫剧启动创建）

| 情况 | 操作 |
|------|------|
| **三个引擎都删了**（建议连同连线一起删） | **漫剧启动** → **开始生成（大纲+角色+分镜）**，会重新创建并跑 LLM |
| **只删了其中一部分** | 可能产生 **重复节点**；建议三个一起删干净再点开始，或从工具栏手动拖入缺失类型 |

删完后若按钮仍显示「重新生成」，以画布上是否还能看到三个引擎为准；没有则会走 **重新创建**。

### 3.2 角色三视图

- 删除某角色的三视图节点 → **角色设定 → 三视图** → 勾选该角色 → **生成选中**。

### 3.3 分镜图 · 镜N

- 删除某镜分镜图 → **分镜脚本 → 分镜图** → 勾选镜号 → **创建选中**（重新创建并带入对白等）。
- 节点还在、只缺对白 → 全选镜号 → **创建选中** 即可 **补写**，无需先删。

### 3.4 视频 / 配音节点

- 删除后 → 对应 **分镜图 · 镜N** → **生成视频** / **生成对白**，会自动 **重新创建** 媒体节点再提交任务。

**提示**：删除后稍等画布 **自动保存**（PATCH 完成）再点生成，更稳妥。

## 4. 常见修补操作

### 4.1 分镜图缺少对白或视频提示

**方式 A（批量，推荐）**

1. **分镜脚本 → 分镜图** Tab。
2. 勾选要对白的镜号 → **创建选中**。
3. 已有镜号会 **补写** `frameDialogue` / `frameVideoPrompt`，不重复建节点。

**方式 B（单镜）**

1. 打开 **分镜图 · 镜N → 生成对白**（或生成视频）Tab。
2. 打开弹窗时会从分镜表 **自动补写并显示**。

### 4.2 某镜 LLM 文案不满意

打开对应引擎（大纲 / 角色 / 分镜）→ **文案** Tab → 改 Prompt → **重新生成** 或 **强制跳过缓存**。

### 4.3 分镜静帧 / 三视图要重画

打开节点弹窗 → 选模型 → **重新生成**（或 **强制跳过缓存**）。同一节点重复点击不会重复入队。

### 4.4 误点「运行全部」

工具栏 **运行全部** 已改为 **按拓扑顺序** 执行，避免多镜同时打满 API；单镜操作仍用各节点底部按钮。

## 5. 节点底部按钮速查

| 节点 | 底部按钮 | 打开弹窗 Tab |
|------|----------|--------------|
| 故事大纲 | 文案、预览 | 文案 / 预览 |
| 角色设定 | 文案、三视图 | 文案 / 三视图 / 预览 |
| 分镜脚本 | 文案、分镜图 | 文案 / 分镜图 / 预览 |
| 分镜图 · 镜N | 分镜图生成、生成视频、生成对白、视频+对白 | 四个 Tab 一一对应 |

节点本体只显示 **摘要 + 状态**；长表单与模型选择均在 **弹窗** 内完成。

## 6. 快捷键（画布通用）

| 操作 | 方式 |
|------|------|
| 框选多节点 | 拖空白处 |
| 平移画布 | 中键 / 右键拖动 |
| 缩放 | 滚轮 / 触控板 |
| 多选 | ⌘ 或 ⇧ + 点击 |
| 撤销 / 重做 | ⌘Z / ⌘⇧Z |
| 删除选中 | Backspace / Delete |
| 拖图片文件入画布 | 自动创建图片节点并上传 OSS |
| 从顶部工具栏拖 icon | 创建对应类型节点 |

选中 ≥2 个节点时，选区上方出现 **分组 / 自动整理 / 删除**。

---

## 7. 删除节点注意事项

产品规则：**删除库条目、画布资源等破坏性操作** 在界面内需 **连续两次确认**；若涉及 OSS 云端文件，第二次确认会说明不可恢复。

---

## 8. 与 story-web 的关系

- canvas-web 漫剧画布与 **story-web 数据不互通**。
- 导出 ZIP 后在本地 / 剪映继续剪辑；不在 story-web 个人空间自动同步。

---

## 9. 故障排查简表

| 现象 | 建议 |
|------|------|
| Provider 列表无 DeepSeek | 确认 book-mall `.env.local` 有 `DEEPSEEK_API_KEY`，重启 `pnpm dev:all` |
| 创建分镜图被拦截 | 按提示到 **角色设定 → 三视图** 补做 **该镜出镜角色** |
| 生成对白无文本 | 分镜表 **台词 / 对白** 列是否有内容；用 **创建选中** 或打开对白 Tab 补写 |
| 429 / 限流 | 避免同时点多镜；用节点单镜生成，勿频繁 **运行全部** |
| KIE 余额不足 | 充值或换 DeepSeek 等非 KIE 模型做文案 |
| 旧画布有预览小节点 | 打开时自动清理；用各引擎 **预览** Tab 代替 |
