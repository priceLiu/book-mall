## 1. 用户操作路径

```text
输入主题 → 创作剧本（大纲）→ 审阅修改并保存 → 定稿生成工作流 → 各列手动生成媒体 → 剪映导出
```

| 步骤 | 节点 | 您做什么 | 系统做什么 |
|------|------|----------|------------|
| 1 | **故事主题** | 填主题、选 LLM，点 **创作剧本** | 创建故事大纲，生成大纲文案 |
| 2 | **故事大纲** | 审阅、修改、**保存**（角色/分镜/对白按需） | 定稿前可随时改；**尚未**创建媒体列 |
| 3 | **故事大纲** | 点 **定稿生成工作流** | 锁定大纲文案，创建下游节点并 **拆分** 文本（不自动生成图/视频） |
| 4 | 媒体列 | 单镜或底栏 **全部生成** | 仅响应您的操作 |
| 5 | **剪映导出** | 导出 ZIP | 读本套视频/配音 |

---

## 2. 定稿规则（简要）

- **定稿** = 点击 **「定稿生成工作流」**；此前可随时编辑保存。
- **定稿后** 故事大纲 **不可再改**（只读审阅）。
- **大纲非空即可** 定稿；角色/分镜/对白 **不强制填满**；无角色时角色列仍创建、可为空。
- **删除** 本套角色/分镜/视频/剪映节点后 → 大纲 **恢复可编辑** → 可改稿并再次 **定稿生成工作流**。
- 故事主题在定稿后会锁定；删列后恢复 **创作剧本**。

## 3. 节点与尺寸

| 节点 | 尺寸 |
|------|------|
| 故事主题 / 故事大纲 | 1020 × 1200 |
| 角色 · 三视图 | 560 × 2100 |
| 分镜 · 分镜图 | 1080 × 2100 |
| 分镜 · 视频 | 500 × 2100 |
| 剪映导出 | 400 × 280（与主题/大纲 **顶边齐平**） |

## 4. 多工作流

一张画布可放多套或多条相同流程，互不干扰；仅 **手动接线** 会跨套连接。

详见 [story-workflow-canonical.md](./story-workflow-canonical.md) §2.4、§6。
