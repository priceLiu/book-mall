# 分镜视频 1.0 · 标准工作流

## 适用

- 新建项目 → 选择「分镜视频 1.0」模板
- 画布仅含 `sbv1-image` 与 `sbv1-video-engine`

## SOP

1. **准备参考图**  
   - 画布空白粘贴多张图片，或工具栏「添加图片」  
   - 每张图为独立 `sbv1-image` 节点

2. **连线**  
   - 从图片节点拖到视频引擎（或引擎 `in_ref` ← 图片 `image`）  
   - 引擎 prompt 区 ContextBar 出现编号 chip

3. **编写 prompt**  
   - 在视频引擎文本框输入描述  
   - 可用 `@图片1` 引用连线顺序

4. **配置工具条**  
   - 创作类型：视频生成  
   - 模型：即梦 Seedance 系列  
   - 参考模式：全能参考 / 首尾帧 / 智能多帧  
   - 比例、时长（或 smart_multi 下分辨率）

5. **生成**  
   - 点击 ↑ 提交  
   - 等待任务完成；结果在引擎节点内预览

6. **调整**  
   - chip 上 X 仅断开连线，图片节点保留  
   - 可重新连线或换模式后再次生成

## 禁止

- 向 sbv1 画布添加 Pro / Pro2 / 漫剧节点  
- 使用 HTTP 访问主站（须 HTTPS，见 deploy 故障速查）
